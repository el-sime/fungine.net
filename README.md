# Fungine.net

A basic game engine for learning porpuses.

## 0 - Intro and setup

- Install dotnet
- Install Monogame template
- `dotnet new sln`
- `dotnet new mgdesktopgl --name fungine`
- `dotnet new xunit --name fungine.test`
